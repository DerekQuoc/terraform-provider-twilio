# TaskRouter Task Channel

This example provisions the following resources:

- taskrouter workspace
- taskrouter task channel

## Requirements

| Name      | Version |
| --------- | ------- |
| terraform | >= 0.12 |

## Providers

| Name   | Version |
| ------ | ------- |
| twilio | n/a     |

## Inputs

No input.

## Outputs

| Name         | Description                           |
| ------------ | ------------------------------------- |
| task_channel | The Generated TaskRouter Task Channel |
| workspace    | The Generated TaskRouter Workspace    |
