# SIP Trunk Origination URL

This example provisions the following resources:

- SIP Trunk
- SIP Trunk Origination URL

## Requirements

| Name      | Version  |
| --------- | -------- |
| terraform | >= 0.13  |
| twilio    | >= 0.4.0 |

## Providers

| Name   | Version  |
| ------ | -------- |
| twilio | >= 0.4.0 |

## Inputs

No input.

## Outputs

| Name            | Description                   |
| --------------- | ----------------------------- |
| origination_url | The SIP Trunk Origination URL |
| trunk           | The SIP Trunk                 |
