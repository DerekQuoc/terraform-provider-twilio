# Serverless Build

This example provisions the following resources:

- serverless service
- serverless function
- serverless function version
- serverless build

## Requirements

| Name      | Version |
| --------- | ------- |
| terraform | >= 0.12 |

## Providers

| Name   | Version |
| ------ | ------- |
| random | n/a     |
| twilio | n/a     |

## Inputs

No input.

## Outputs

| Name     | Description                       |
| -------- | --------------------------------- |
| build    | The Generated Serverless Build    |
| function | The Generated Serverless Function |
| service  | The Generated Serverless Service  |
